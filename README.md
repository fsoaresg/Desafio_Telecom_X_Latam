# Desafio_Telecom_X_Latam
Telecom X enfrenta una alta tasa de cancelación de clientes (churn), lo que impacta directamente en sus ingresos y sostenibilidad.   Este proyecto tiene como objetivo analizar los factores que influyen en la evasión de clientes, utilizando Python y técnicas de Análisis Exploratorio de Datos (EDA), con el fin de generar insights accionables.
